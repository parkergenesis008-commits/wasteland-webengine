## Phenomenon

The Bravyi-König bound states that 2D topological stabilizer codes **cannot** support transversal non-Clifford gates — a fundamental limitation on fault-tolerant quantum computation. This paper extends the bound to **Clifford Hierarchy (CH_D) stabilizer codes** in n dimensions, proving that:

\[
\text{CH}_D \text{ codes in } n \text{ dimensions:} \quad \text{transversal gates at level } D+1 \text{ require } n \geq D+1
\]

For the most interesting case — **CH₂ codes** (those supporting transversal T gates, the 3rd level of the Clifford Hierarchy) — the constraint is \(n \geq 3\). Three-dimensional codes can implement **transversal T gates**, while 4D codes can implement transversal CCZ gates (4th level).

**Magic state distillation** provides the resource: by preparing noisy copies of the magic state \(|A\rangle = T|+\rangle\) and distilling them through the CH_D code's error detection properties, one achieves universal fault-tolerant computation with overhead scaling \(O(\log^\gamma(1/\varepsilon))\).

The key mathematical structure: **Z_X symmetries** and linear constraints classify which logical gates are transversally implementable. The symmetry group of the code determines the gate set — a direct mapping from **code geometry to computational power**.
